-- ЧТО ДЕЛАЕТ ПРОЕКТ:

Сайт для отображения информации по заявкам и обращениям на
электро-присоединение, а также отображения заявок на аварийно
востановительные работы.

Доступ к заявкам и обращениям на электро-присоединение ограничен. Добавить
новых пользователей можно с момощью db_users.db в папке database.
